Projecte de Unity per practicar l'execució d'animacions a partir de codi i de l'Animator
Se'ns va proporcionar l'art del personatge.
L'objectiu va ser a partir de l'animació del porjecte_1 fer que es reprodueixin les animacions correctament a partir dels inputs i que es rescatés la coherència dels estats en l'animació.

Vaig adjunta un script C# de MonoBehaviour i a partir de la classe Input vaig assosiar tecles a accions del personatge:
- W: Salt
- A: Caminar
- A + Shift/Mayus: Correr
Aquestes accions van lligades a estats del animator i es relacionen entre elles.

El projecte s'ha obert prèviament en la versió 2022.3.30f1. Desconec si es pot obrir en altres versions.
Es pot arrossegar la carpeta ART a un projecte buit i el prefab 
a una escena.
